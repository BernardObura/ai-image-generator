## 🚀 Quick Start Guide

### 1. **Extract and Setup**
```bash
# Extract the project
tar -xzf ai-image-generator-react-final.tar.gz
cd ai-image-generator-react

# Install dependencies
npm install
```

### 2. **Configure API Key**
```bash
# Copy environment template
cp .env.example .env

# Edit .env and add your Hugging Face API key
# VITE_HUGGINGFACE_API_KEY=your_api_key_here
```

### 3. **Start Development**
```bash
# Start development server
npm run dev

# Open http://localhost:5173
```

## 🌐 Deployment Options

### **Option 1: Vercel (Recommended)**
1. Push to GitHub repository
2. Connect to Vercel
3. Add API key as environment variable
4. Deploy automatically!

### **Option 2: Netlify**
1. Build: `npm run build`
2. Upload `dist` folder
3. Configure environment variables

### **Option 3: GitHub Pages**
1. Push to GitHub
2. Enable GitHub Pages with Actions
3. Add API key to repository secrets

## 🔑 API Key Setup

### **Get Hugging Face API Key:**
1. Visit: https://huggingface.co/settings/tokens
2. Create new token with "Read" permissions
3. Copy and add to `.env` file
